Detailed Description

Facade of Box2D entity. This class corresponds to all our physical elements. They are store in the array m_physicalEntityArray, contains in the EntityManager.

void Symp::PhysicalEntity::setCustomChainHitbox ( const char *  collisionFileName)

Set a custom hitbox of the physical entity (from json file, created by the software Box2D-editor). The previous hitbox (a polygon shape with 4 vertices), created in the constructor, is replaced by the new one. Warning : The collision between b2ChainShapes is not supported in Box2D. So only the dino has a b2ChainShape.
