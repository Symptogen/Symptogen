Detailed Description

The MetaEntity is used to store the information about one entity during the parsing of it. The reason of the existence of this strut is because the ParserLevel uses a visitor pattern to parse the data, so the different informations we get from different XML elements needs to exists over the process. When both entities (physical and render) are created, the MetaEntity is reseted to be ready to save datas of the next entity.
