Detailed Description

Facade of IND_Entity2d. This class corresponds to all our render elements, visible in the game. They are store in the array m_renderEntityArray, contains in the EntityManager. Warning : in the menu, we use GuiComponent (an other class, and an other facade of IND_Entity2d). It provides tools to :

  • Set a surface
  • Set an animation
    See Also
    EntityManager
