enum Symp::LayoutFillAttribute

enum The LayoutFillAttribute enum presents convenient fields for the choice of the type of #Layout that will be used. The customization of a #Layout can be made following these three types : make a border, or make a background, or do not show the #Layout. When a #Layout is used with the border or background customization type selected, then the unique #Color given in argument is used wether for the border, or the background. Impossible to use both.
