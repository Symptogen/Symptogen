Detailed Description

GuiComponent facade for the creation of Menu graphical entities The GuiComponent class implements a facade of the Indielib IND_ENtity2d elements for the unique use of the menus. Each graphical element that compose a menu inherits from the GuiComponent class. The elements are stored in the #MenuManager class and rendered/updated by the #GameManager class. In order to use the Indielib architecture, this class implements static method for the IND_ImageManager, IND_SurfaceManager and the IND_FontManager initialization destruction.
