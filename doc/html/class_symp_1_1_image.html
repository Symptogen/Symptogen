Detailed Description

This class inherits the GuiComponent class. A Image class display a texture. As an Image got the inherited parameters of size, the update is made following the type of ratio choosen for the Image. The types of ratio are defined in the AspectRatio enum. The simpliest way to use Image is with the #Layout class, that handle the position and the size of the Image. of Image that will be rendered.

Image class constructor inherits the GuiComponent class Responsible for the initialization of the class private attributes. This constructor will associate a texture to its Indielib IND_Entity2d. The position and scale atributes have default values that can be left for using the Image with a #Layout.

Parameters
filePathis the path to the image file
iPosXthe x coordinate of the upper-left corner of the Image in pixels (default = 0)
iPosYthe y coordinate of the upper-left corner of the Image in pixels (default = 0)
iScalethe uniform scale that will be applied for both width and height between 0 and 1 (default no scale = 1)
